# ReactTesting
After cloning the repo:
  - Open new terminal
  - "cd" into "backend" folder
  -  "npm start" should start the server
  - Open another terminal 
  -  "cd" into "front_end" folder
  -  "npm start" should start the frontend react at localhost:3000
